
# FIAP - Pós Tech -- Curso IA para Devs

## Tech Challenge -- Fase 1 - Welcome to IA para Devs

Este projeto visa desenvolver a base de um sistema inteligente de suporte ao diagnóstico para um hospital universitário.

A solução utiliza técnicas de Inteligência Artificial e Machine Learning para analisar automaticamente exames médicos e documentos clínicos, auxiliando a triagem, destacando informações relevantes e apoiando a tomada de decisão médica, com foco em eficiência, redução de erros e otimização do tempo das equipes de saúde.
